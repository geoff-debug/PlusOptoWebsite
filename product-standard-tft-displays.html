Standard TFT Displays

High resolution, vibrant full colour TFT display panels from Winstar Display, available in sizes from 1.77" to 10.4" in resolutions from QVGA to WXGA. Now an affordable option for a wide range of applications, our standard TFT range offers low minimum order quantities and is suitable for long-term design-in to industrial applications. Options include touch panel overlays, controller boards, SPI interface, high brightness and ultra wide temperature variants.

3.5" Standard TFT Display

WF35YTIACDNT0 is a 3.5-inch resistive touch TFT-LCD module with a resolution of 320×240 pixels, a brightness of 300 cd/m², and a typical contrast ratio of 350:1. The key feature of the WF35Y series is the inclusion of an integrated and standardized 40-pin FPC, supporting both DE and SYNC modes.

4.3" Standard TFT Display

The WF43GTIAEDNT0 TFT LCD module is a 4.3-inch diagonal display with a resistive touch panel (RTP) and a resolution of 480x272 pixels, offering vibrant full-color output. With outline dimensions of 105.5 x 67.2 mm and an active area of 95.04 x 53.86 mm, it provides ample screen real estate for various applications. This module is built with the ILI6480B IC and supports a 24-bit RGB interface.

View Details →

5.7" Standard TFT Display

The 5.7-inch color TFT LCD display module is designed for industrial applications and features the HX8218+HX8615 driver IC, supporting an 18-bit RGB interface. The module is equipped with a built-in power board that provides stable power management, simplifying the system's power configuration.

View Details →

7.0" Standard TFT Display

WF70B4TIAGDNN0 is a color TFT LCD module with a resolution of 800x480 pixels and a diagonal size of 7 inches. The module dimensions are 165 x 100 mm, and the active area size is 154.08 x 85.92 mm. This TFT model is built-in with HX8264 and HX8664 driver ICs. WF70B4TIAGDNN0 supports a 24-bit RGB interface.

View Details →

8.0" Standard TFT Display

The WF80BTIAGDNNC is an 8-inch TFT LCD display module with a resolution of 800x480 pixels, an aspect ratio of 16:9, a brightness of 420 nits (typical value), and a contrast ratio of 500:1. The module has an active area size of 176.64 x 99.36 mm and an outline dimension of 192.80 x 116.90 mm. The display uses EK9713 and EK73002 driver ICs, supporting 24-bit RGB interfaces.
