Smart Displays

Out-of-the-box display solutions supporting UART, RS485, CAN OPEN and Custom CAN ID protocols.

Overview

The "Winstar Smart Display Series" is an innovative display product offering shorter time to market and ease of implementation. The current range includes CAN, RS485, UART protocols as well as entry-level interfaces.

Our Smart Display range includes:

- Colour TFT Smart Displays: Standard modules with UI objects, supporting CAN/UART/I2C/SPI/USB/WiFi/BT interfaces

- UART TFT Smart Displays: High-performance solution with real-time simulation, zero-code drag & drop design, supporting TouchGFX, LVGL, QT and Embedded Wizard GUI frameworks More Information

- CANopen TFT Series: CAN bus interface with CANopen & Custom CAN ID protocols More Information

- RS485 Modbus Smart Display: Modbus protocol, pre-loaded with standard UI objects More Information

- 5" CAN bus TFT Smart Display: Slave device controlled via CAN bus command

- 7.0" CAN Bus TFT Smart Display: With PCAP Touch, USB2CAN/MCU/Raspberry Pi compatible

- 3.9" Bar-Type UART Smart Display with GPIO: IPS TFT with capacitive touch

- 4.3" and 5.0" RS485 Smart Display: 8-channel bi-directional GPIO interface

- 7.0" RS485 Modbus Smart Display with PCAP: IPS TFT, multiple host platform support

- 7.0" IPS UART Smart Display with GPIO Touch: 1024x600 resolution

- 10.1" IPS UART Smart Display with GPIO Touch: Largest size in the range

- Dustproof Cap Touch TFT Smart Display: 7" IP54-rated with CAN bus communication for industrial environments

Wide-temperature options available for automotive, marine, power generation and oil-and-gas applications.
