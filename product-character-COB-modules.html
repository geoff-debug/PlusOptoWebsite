Character COB LCD Modules

WINSTAR offers a wide range of Character LCD Display Modules, built with COB (Chip on Board) structure for enhanced reliability and compact design. Available in TN, STN, FSTN, FFSTN and display formats from 8x2 to 40x4, featuring 5x8 dot matrix characters for clear and crisp display.

8x2 Character COB LCD Module

The WH0802A1 is an 8-character by 2-line LCD display module (8x2 Character LCD Display Module) with a module dimension of 58.0 x 32.0 mm and an active area size of 27.81 x 11.5 mm (1.18" diagonal). It operates with a 5V power supply, compatible with 3V, while the negative voltage is limited to 3V. The 8x2 LCD module comes with a PCB board that includes mounting holes, facilitating easy installation in various applications.

12x2 Character COB LCD Module

The WH1202A model is a 12-character by 2-line small/miniature LCD display with outline dimensions of 55.7 x 32 mm, an active area size of 37.85 x 11.7 mm, and a diagonal size of 1.56 inches. The default power supply is 5V, and there is an optional 3V configuration. If opting for the 3V module, a negative voltage version is also available. The 12x2 LCD module comes with a PCB board that includes mounting holes, facilitating easy installation in various applications.

View Details →

16x1 Character COB LCD Module

Model WH1601 is a 16x1 Character LCD Display. This display is equipped with the ST7066 controller and features a default interface of 6800 4/8-bit parallel. Additionally, it can be configured with SPI and I2C interfaces using the RW1063 controller IC. The LCD Display 16x1 module comes with a PCB board that includes mounting holes, facilitating easy installation in various applications. It operates with a 5V power supply, compatible with 3V, while the negative voltage is limited to 3V.

View Details →

16x2 Character COB LCD Module

The WINSTAR WH1602A is a dot matrix LCD display module capable of displaying 16 characters by 2 lines, featuring a Chip on Board (COB) design and a 2.26-inch diagonal screen size. It is equipped with the ST7066 controller IC by default, which supports the 6800 4/8-bit parallel interface. Optional interfaces such as SPI and I2C are available using the RW1063 controller IC.

View Details →

16x4 Character COB LCD Module

The WINSTAR WH1604A is a 16x4 character LCD module designed with a standard COB structure and ST7066 controller IC. Its default interface is 6800 4/8-bit parallel, and it is also available with SPI and I2C options using the RW1063 controller IC. The module operates on a +5V power supply and is also available with a +3V option. With a display size of 2.36 inches, the WH1604A series is suitable for various system integrations.

View Details →

20x2 Character COB LCD Module

The WINSTAR WH2002A display LCD 20x2 module series features a standard COB structure and an ST7066 controller IC. Its default interface is 6800 4/8-bit parallel, with optional SPI and I2C interfaces available using the RW1063 controller IC. The backlight LEDs can be driven via pin 1, pin 2, pin 15, pin 16, or A/K connections. The modules operate over a temperature range of -20℃ to +70℃ and can be stored at -30℃ to +80℃. The display size is 2.93 inches, suitable for various system integrations. The modules are powered by +5V (also available with +3V); negative voltage operation is only supported when using the +3V supply.

20x4 Character COB LCD Module

WINSTAR WH2004A is a 20x4 character LCD display built with the ST7066 controller IC. Its default interface is 6800 4/8-bit parallel. The series also offers SPI and I2C interfaces using the RW1063 controller IC. WH2004A modules are available with multiple backlight colors, including blue, green, white, yellow-green, amber, red, and RGB LEDs. The ST7066 IC provides multiple character font options, including English/Japanese, English/West European, English/Scandinavian European, English/Cyrillic (Russian), and Hebrew/Arabic.

24x2 Character COB LCD Module

WH2402A is 24 characters by 2 lines LCD 24x2 display module which is built in with ST7066 controller IC; its default interface is 6800 4/8-bit parallel. This LCD display 24x2 model WH2402A is available for without backlight or with LED backlight options. This display LCD 24x2 modules is available in various LCD panel colors including gray, yellow/green and blue LCD options. The LEDs can be driven by pin 1, pin 2, or pin 15 pin 16 or A/K. There are different character fonts/texts available for ST7066 IC including English/Japanese, English/West European, English/Scandinavian European, or English/Cyrillic (Russian) as options.

40x2 Character COB LCD Module

WINSTAR WH4002A is 40 characters by 2 rows of 40x2 LCD display module which is built in with ST7066 controller IC; its default interface is 6800 4/8-bit parallel and 5V or 3.3V power supply optional. This display LCD 40x2 model WH4002A is available for without backlight or with LED backlight options. This LCD display 40x2 is available in various LCD panel colors including gray, yellow/green and blue LCD options. This display LCD 40x2 module can be operating at temperatures from -20℃ to +70℃; its storage temperatures range from -30℃ to +80℃, its built in character set supports English/Japanese, English/West European, English/Scandinavian European, or English/Cyrillic (Russian) options.
