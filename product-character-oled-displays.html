Character OLED Displays

Standard character OLED displays facilitate simple upgrade from traditional TN and STN displays and utilise the same mechanical size and pin-out as these traditional displays. The full range are supplied with an integrated "4 in 1" font driver allowing the user to select from English, Japanese, European or Russian characters. We also offer OLED displays as a mechanically compatible solution to traditional vacuum fluorescent (VFD) displays.

Character OLED Displays - Full Range

Our character OLED modules come in standard sizes and can be used as compatible replacements for STN LCD modules. Our OLED character displays are available in 8x2 OLED, 12x2 OLED, 16x1 OLED, 16x2 OLED, 20x2 OLED, 20x4 OLED, and 40x2 OLED format with 5x8 dot matrix characters.

8x2 Character OLED Display

The WEH000802A is an 8x2 OLED display module with an outline size of 58.0 × 32.0 mm and a viewing area (VA size) of 38.0 × 16.0 mm, sharing the same pinout as the Winstar STN LCD item no. WH0802A. It features OLED technology with 8 characters x 2 lines and a diagonal size of 1.2 inches.

16x1 Character OLED Display

WEH001601B is a COB structure 16x1 OLED (16 characters × 1 line) display module with a diagonal panel size of 2.25 inches. The module dimensions measure 80.0 × 36.0 mm, with an active area size of 56.95 × 5.55 mm. It integrates the WS0010-TX IC, supporting a default 6800 interface, along with additional options of 8080 and SPI, and operates with a 1/8 driving duty. All Winstar OLED Character COB modules are equipped with the all-in-one Fonts controller IC WS0010-TX, providing 4 font sets in one device. The built-in ROM standard font library includes ASCII, English, Russian, European, and Japanese character sets.

16x2 Character OLED Display

WEH001602A is a COB character OLED module with a PCB board, featuring a diagonal size of 2.29 inches and displaying 16 characters x 2 lines. This OLED shares the same outline size (80.0 x 36.0 mm), viewing area size (66.0 x 16.0 mm), and pinout as Winstar LCD item no. WH1602B. We recommend customers consider using this low-power 16x2 character OLED to replace existing STN character displays or for new projects.

20x2 Character OLED Display

The WEH002002A is a COB-structured OLED display module featuring a 20x2 character configuration. Supporting the default interface of 6800, with optional 8080 or SPI interfaces, it operates in a 1/16 duty drive mode. All standard Winstar COB OLED character types are equipped with this all-in-one Fonts controller IC, offering support for ASCII, English, Russian, European, and Japanese character sets via a built-in ROM standard font library.

40x2 Character OLED Display

The Winstar WEH004002A is a 5.85-inch OLED module characterized by a COB structure, with the same outline size, VA size, and pinout as the Winstar STN LCD standard item WH4002A.
