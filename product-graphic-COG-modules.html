Graphic COG LCD Modules

Our Graphic COG LCD modules (Chip-on-Glass) monochrome displays provides graphic type in resolution of 128x64 to 320x240. The COG package offers high quality with the benefits of light weight, and low power consumption. A connection board is available to change the interface from FPC to pins or cables. with a variety of pin out connections to suit your requirements. The COG LCM technology is suitable for the applications that demand more compact package displays.

128x64 Graphic COG LCD Module

WO12864A1 is a 2.2 inch 128x64 COG monochrome Graphic LCD module. This LCD module is built in with ST7565P controller or equivalent IC. It supports 6800 8-bit, 8080 8-bit parallel and SPI interface. This LCD module can be operating at temperatures from -20℃ to +70℃; its storage temperatures range from -30℃ to +80℃. This LCD COG 128x64 display is very suitable for the applications that demand more compact package displays.

128x128 Circular LCD Module

The WO128128A is a COG monochrome graphic circular LCD display module with a 128x128 dot matrix format. This Circular LCD screen display module is equipped with the ST75161 IC and supports 8080/6800 parallel, 3-wire and 4-wire serial SPI, and I2C interfaces. The viewing direction is set at 6 o'clock, with a 1/136 driving duty and 1/12 BIAS.

View Details →

132x32 COG LCD Module (FPC)

WO13232A is a monochrome COG graphic LCD module made of 132 x 32 dots matrix with FPC pinout. This COG module is built-in with ST7567A IC; it supports I2C interface, viewing direction 6 o'clock, 1/33 duty, and 1/6 Bias. WO13232A module has the same outline dimension size as WG12232B / WG12232G (with PCB) but with a higher resolution.

240x128 COG Graphic LCD Module

WO240128A is a 3.75-inch monochrome graphic LCD module featuring a COG (Chip on Glass) structure and a resolution of 240x128 dots. It is equipped with the UC1608 controller IC and supports multiple interface options, including 6800, 8080, 3-wire SPI, and 4-wire SPI. This module is lightweight, energy-efficient, and ideal for applications that require high display quality and space efficiency, such as industrial instruments, medical devices, and handheld equipment.
